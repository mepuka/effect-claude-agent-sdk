# Examples

## Prerequisites

- Bun 1.0+
- `ANTHROPIC_API_KEY` set in your environment

```sh
export ANTHROPIC_API_KEY=your-key
```

## Running

```sh
bun examples/01-hello-world.ts
```

## Learning Path

1. **01-hello-world.ts** – Minimal query
2. **02-streaming-text.ts** – Stream text chunks
3. **03-custom-tool.ts** – Define and use a tool
4. **04-multi-turn.ts** – Multi-turn conversation
5. **05-error-handling.ts** – Catch and handle errors
6. **06-persistence.ts** – Persist chat history to disk
7. **07-hooks.ts** – Hook presets for logging
8. **08-sync.ts** – Remote sync wiring
